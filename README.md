SIS-11 (Spatial Impossibility System)
Author: Sitiryampi
SIS-11 is a formal system based on ZFC, designed to study limits of definability via 11-dimensional formula encoding. It introduces the notion of “11D-definable” numbers, where a formula φ(x₁…x₁₁) of bounded length encodes a unique vector of 11 syntactic parameters.
The system includes a hierarchy SISₙ → SIS_ω, which allows the construction of meta-diagonalizing numbers that cannot be defined at any finite stage. One such number is K_KKK (Kafka’s Kitty Scratcher), defined as exceeding all 11D-definable numbers within the system.
